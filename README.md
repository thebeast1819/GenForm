## Prerequisites

Before setting up the app, ensure the following dependencies are installed:

- **Node.js** (v14.x or above)  
  [Download Node.js](https://nodejs.org/)
- **npm** or **yarn** (npm comes with Node.js)
- Git (optional, for version control)
